A list of things to do
Not so long ago, I went to Reddit to ask a question to the world: What is something no one has ever done before?
The original purpose of this was to list things no one had ever done before, and therefore, if you were ever bored, you could run through this list and try to do some of them.

Here are the best replies so far, in my humble opinion.

  • Audio

    Equalizer? Audio manipulation? Live js canvas representation of a song?

  • Live Self-Writing Watch

    A watch that writes the time, live, in MSPaint or similar. Perhaps in js canvas.

  • Intelligence Generator

    Input a text. Press generate. Your text is automatically made smarter.

  • Dynamic Wikipedia Map

    Every single page in Wikipedia linked in a 3D environment by strings that represent similarity / relation.

  • Session Logger

    Displays last time pc was turned on, as well as a key logger of that session.

  • Everything App

    Displays time, date, weather, world map, info on windows opened, etc

  • Mobile Game Maker

    Engine for building games on mobile, without requiring any coding abilities.
